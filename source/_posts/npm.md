#### 本地打包
	npm pack [<directory || current_directory>] 本地打包，生成package
	npm install <package_name> 安装本地包

#### 在线打包
	npm addUser //绑定账号
		Username: your name
		Password: your password
		Email: yourmail@gmail.com
	npm whoami //查看当前账号
	npm publish //发布
